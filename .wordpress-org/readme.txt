Integrieren Sie eine Liste von Trainings der Plattform [sicherheitstraining24.de](https://www.sicherheitstraining24.de/). Das Plugin erstellt filterbare Listen, aus denen Sie direkt auf die Detail und Buchungsseite des jeweiligen Trainings auf [sicherheitstraining24.de](https://www.sicherheitstraining24.de/) verlinken können. Die Tabellenansicht kann nach Ihren Bedürfnissen angepasst werden. Es können mehrere Traingsplätze zu einer Gesamtansicht zusammengeführt werden.


== Beschreibung ==

* Integrieren Sie eine Liste von Trainings der Plattform [sicherheitstraining24.de](https://www.sicherheitstraining24.de/). 
* Das Plugin erstellt filterbare Listen, aus denen Sie direkt auf die Detail und Buchungsseite des jeweiligen Trainings auf [sicherheitstraining24.de](https://www.sicherheitstraining24.de/) verlinken können. 
* Die Tabellenansicht kann nach Ihren Bedürfnissen angepasst werden. 
* Es können mehrere Traingsplätze zu einer Gesamtansicht zusammengeführt werden.
* Das Plugin wird über den Shortcode `[sht24]` genutzt.
* Die Buchung erfolgt über die Webseite [sicherheitstraining24.de](https://www.sicherheitstraining24.de/).

== Ersteller ==

Das Plugin wurde erstellt von: [Verkehrswacht Siegerland-Wittgenstein e. V.](https://www.verkehrswacht-siegerland.de/) / [Bartzik Webdesign](https://webdesign.bartzik.net/)

== Installation ==

1. Laden Sie das Plugin in das Verzeichnis `/wp-content/plugins/`
1. Aktivieren Sie das Plugin über das Adminmenü 'Plugins'
1. Nehmen Sie die notwendigen Plugin-Einstellungen vor
1. Nutzen Sie den Plugin-Shortcut `[sht24]` in Ihren Seiten (weitere Details auf der Plugin-Einstellungsseite in der WordPress-Administration)
